### Task 1 — Threat Intelligence Report (2024–2025)

**Status:** ✅ Completed  
**Deliverable:** Professional Threat Intelligence Report (Word Document)

#### Objective
Analyze the current cybersecurity threat landscape and present findings in a structured, professional report — simulating the day-to-day output of a Cybersecurity Analyst Intern.

#### What's Covered

| # | Threat | Category |
|---|--------|----------|
| 1 | AI-Powered Phishing & Deepfake Social Engineering | Social Engineering |
| 2 | Ransomware-as-a-Service (RaaS) | Malware / Criminal Ecosystem |
| 3 | Cloud Security Misconfigurations | Cloud Security |
| 4 | IoT Vulnerabilities & Botnets | Network / Hardware |
| 5 | Zero-Day Exploits | Vulnerability Research |

#### Report Sections
- Introduction to Cybersecurity (what it is, why it matters, 2024–25 landscape stats)
- 5 Threat Deep-Dives — each with impact analysis, real-world case study, and 3–4 preventive measures
- Threat Summary Table
- Conclusion & Future Scope
- References (12 cited sources)

- **Deepfake BEC Attack (2024)** — $25M wire transfer fraud using AI-generated video call of a CFO
